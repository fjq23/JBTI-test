# JBTI 测试

JBTI 全称是 `Joke Bored Personality Test`，也可以理解成 `Joking But True`。

这是一个纯静态单页测试页，当前版本包含：

- 30 道题
- 16 个常规人格
- 2 个隐藏结果：`DJB`、`XJB`
- 新版海报占位逻辑
